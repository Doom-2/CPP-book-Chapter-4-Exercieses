<a name="bleep-the-word"></a>
>Скрыть слово

Программа, «заглушающая» нежелательные слова. \
Вы вврдите несколько слов, нежелательные заменяются словом BLEEP.
<br/><br/>

<a name="median-temp"></a>
>Медианная температура

Введите несколько значений температур и среднее значение,\
когда половина значений больше него, другая половина меньше.\

Особенности:
- расчет как для нечетного, так и четного количества элементов
<br/><br/>

<a name="vector-elems-compare"></a>
>Сравнение элементов вектора типа \<double\>

Введите несколько значение типа \<double\>, как расстояния между 2 соседними городами\
и узнайте сумму всех расстояний, наименьшее и наибольшее и среднее арифметическое.
<br/><br/>

<a name="numbers-guessing-game"></a>
>Игра «Угадай число»

Загадайте число от 1 до 100 и программа угадает его за максимум 7 попыток.

Особенности:
- классический бинарный поиск
<br/><br/>

<a name="simple-calc"></a>
>Простой калькулятор

Введите 2 числа, выберите операцию и получите результат ее вычисления.
<br/><br/>

<a name="spelling-numbers"></a>
>Строковое представление цифры

Ввведите строковое представление цифры 1-9 и получите ее числовое значение.\
Присутсвует и обратная логика - введите цифру и получите ее строковое представление.

Особенности:
- использование регулярного выражения для проверки того, что строка - тольцо цифры
- обработка пустой строки
- использование одного цикла для обработки обоих случаев
<br/><br/>

<a name="advanced-calc"></a>
>Продвинутый калькулятор

Калькулятор с поддержкой строковых представлений операндов и операций.

Особенности:
- циклическая работа программы с выходом по `Esc`
- алгоритм округления чисел типа \<double\> до 2-х знаков\
  с опусканием незначащих нулей и точки, когда возможно.
- обработка исключения «деление на 0»
- обработка некорректного и пустого ввода
<br/><br/>

<a name="grains-on-chessboard"></a>
>Зерна на шахматной доске

Программа вычисляет сколько клеток шахматной доски потребуется для размещения на ней\
определенного количества зерен риса, если класть на каждую следующую клетку доски\
вдвое больше зёрен, чем на предыдущую, начиная с одного.


Особенности:
- отслеживается текущий номер клетки, количество зерен в ней,\
  количество зерен во всех предыдущих клетках
- максимальное количество зерен ограничено вместимостью типа \<int\>
<br/><br/>

<a name="#grains-on-chessboard-max"></a>
> Зерна на шахматной доске (максимумы)

Модификация предыдущей задачи для вычисления количества клеток\
шахматной доски, необходимого для размещения на ней количества зерен,\
эквивалетного максимальным значениям для типов \<int\> и \<double\>.
<br/><br/>

<a name="rock-paper-scissors"></a>
> Игра «Камень, ножницы, бумага»

Консольный вариант популярной игры.\
Сделайте выбор, компьютер, в свою очередь, сделает его рандомно.\
Далее высвечивается результат игры.

Особенности:
- использование конструкции switch-case для обработки ввода пользователя
- обработка некорректного ввода
- зацикливание игры с обновлением экрана по нажатию любой клавиши
- выход из программы по `Esc`
<br/><br/>

<a name="bleep-the-word"></a>
>Скрыть слово

Программа, «заглушающая» нежелательные слова. \
Вы вводите несколько слов, нежелательные заменяются словом BLEEP.
<br/><br/>

<a name="median-temp"></a>
>Медианная температура

Введите несколько значений температур и узнайте среднее значение,\
когда половина значений больше него, другая половина меньше.

Особенности:
- расчет как для нечетного, так и четного количества элементов
<br/><br/>

<a name="vector-elems-compare"></a>
>Сравнение элементов вектора типа \<double\>

Введите несколько значение типа \<double\>, как расстояния между двумя соседними городами\
и узнайте сумму всех расстояний, наименьшее и наибольшее и среднее арифметическое.
<br/><br/>

<a name="numbers-guessing-game"></a>
>Игра «Угадай число»

Загадайте число от 1 до 100 и программа угадает его за максимум 7 попыток.

Особенности:
- классический бинарный поиск
<br/><br/>

<a name="simple-calc"></a>
>Простой калькулятор

Введите 2 числа, выберите операцию и получите результат ее вычисления.
<br/><br/>

<a name="spelling-numbers"></a>
>Строковое представление цифры

Ввведите строковое представление цифры от 1 до 9 и получите ее числовое значение.\
Присутсвует и обратная логика - введите цифру и получите ее строковое представление.

Особенности:
- использование регулярного выражения для проверки того, что строка - тольцо цифры
- обработка пустой строки
- использование одного цикла для обработки обоих случаев
<br/><br/>

<a name="advanced-calc"></a>
>Продвинутый калькулятор

Калькулятор с поддержкой строковых представлений операндов и операций.

Особенности:
- циклическая работа программы с выходом по `Esc`
- алгоритм округления чисел типа \<double\> до 2-х знаков\
  с опусканием незначащих нулей и точки, когда возможно.
- обработка исключения «деление на 0»
- обработка некорректного и пустого ввода
<br/><br/>

<a name="grains-on-chessboard"></a>
>Зерна на шахматной доске

Программа вычисляет сколько клеток шахматной доски потребуется для размещения на ней\
определенного количества зерен риса, если класть на каждую следующую клетку доски\
вдвое больше зёрен, чем на предыдущую, начиная с одного.


Особенности:
- отслеживается текущий номер клетки, количество зерен в ней,\
  количество зерен во всех предыдущих клетках
- максимальное количество зерен ограничено вместимостью типа \<int\>
<br/><br/>

<a name="#grains-on-chessboard-max"></a>
> Зерна на шахматной доске (максимумы)

Модификация предыдущей задачи для вычисления количества клеток\
шахматной доски, необходимого для размещения на ней количества зерен,\
эквивалетного максимальным значениям для типов \<int\> и \<double\>.
<br/><br/>

<a name="rock-paper-scissors"></a>
> Игра «Камень, ножницы, бумага»

Консольный вариант популярной игры.\
Сделайте выбор, компьютер, в свою очередь, сделает его рандомно.\
Далее высвечиваются результат игры.

Особенности:
- использование конструкции `switch-case` для обработки ввода пользователя
- обработка некорректного ввода
- зацикливание игры с обновлением экрана по нажатию любой клавиши
- выход из программы по `Esc`
<br/><br/>

<a name="prime-numbers"></a>
> Простые числа от 1 до 100

Программа находит и печатает все простые числа от 1 до 100.\
Число является простым, если оно делится без остатка только на 1 и на само себя.

Особенности:
- сокращение времени вычислений путем поиска делителей во внутреннем цикле\
  не до текущего числа из внешнего цикла, а до его корня
<br/><br/>

<a name="prime-numbers-in-range"></a>
> Простые числа от 1 до max

Модификация предыдущей задачи для поиска простых чисел в диапазоне от 1 до max.
<br/><br/>

<a name="sieve-of-eratosthenes"></a>
> Решето Эратосфена от 1 до 100

Реализация поиска простых числел от 1 до 100 по алгоритму греческого математика.

Особенности:
- совместное использование функций стандартной библиотеки `std::erase` и `std::remove`\
  для удаления элемента из контейнера по значению с уменьшенем его размера
<br/><br/>

<a name="sieve-of-eratosthenes-in-range"></a>
> Решето Эратосфена от 1 до max

Модификация предыдущей задачи для поиска простых чисел в диапазоне от 1 до max.
<br/><br/>

<a name="first-n-prime-numbers"></a>
> Первые n простых чисел

Модификация задачи «Решето Эратосфена» для поиска введенного количества\
простых чисел, начиая с первого (2).
<br/><br/>

<a name="mode-of-set-of-ints"></a>
> Мода набора положительных чисел

Введите последовательность целых чисел и узнайте число, которое повторяется\
в последовательности наибольшее количество раз.

Особенности:
- алгоритм основан на сравнении соседних чисел отсортированного массива\
  и попутного сохранения / обновления максимального числа повторений
<br/><br/>

<a name="mode-of-seq-of-strings"></a>
> Мода последовательности строк

Модификация предыдущей задачи для поиска наименьшего и наибольшего значений,\
а также моды последовательности строк типа \<string\>.

Особенности:
- использование манипуляторов `setw`, `left`, `right` для форматированного вывода
- сортировка строк с указанием алгоритма (по длине строки) через лямбда-выражение (C++ 11)
<br/><br/>

<a name="quadratic-equations"></a>
> Решение квадратных уравнений ax<sup>2</sup> + bx + c = 0

Введите `a`, `b`, `c` и получите 1 или 2 корня квадратного уравнения.

Особенности:
- поддержка вещественных и комплексных корней 
- проверка на ввод чисел с плавающей точкой
- алгоритм округления чисел типа \<double\> до n знаков\
  с опусканием незначащих нулей и точки, когда возможно.
<br/><br/>

<a name="key-value-table"></a>
> Студенти и оценки

Объеденены упражнения 19-21.\
Введите набор пар, состоящих из имени студента и его количества баллов.\
Имя-баллы разделяйте пробелом, а пары - через `Enter`\
Завершите набор вводом строки `NoName`. Получите результаты на экране.\
Далее введите имя студента, чтобы вывести его количество баллов\
или введите количество баллов, чтобы вывести имена всех студентов с этими баллами.

Особенности:
- использование манипуляторов `setw`, `left`, `right` для форматированного вывода
- проверка, что имя содержит хотя бы одну букву через функции `std::any_of()` и `std::isalpha()`
- проверка вектора строк на дубликаты, завершение программы в случае их наличия
<br/><br/>

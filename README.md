<a name="interactive-letter"></a>
>Интерактивное письмо

Вы пишите письмо вашему другу или подруге. \
Программа задает вопросы и выводит на экран содержание, \
в зависимости от введенных значений.

Особенности:
- проверка на ввод целых чисел
- проверка на ввод только одного символа
- зацикливание некорректного ввода с выходом по `q`
<br/><br/>

<a name="mile-km-conv"></a>
>Конвертор миль в километры

Введите дистанцию в милях и получите вывод в километрах.

Особенности:
- проверка на ввод целых чисел и чисел с плавающей точкой
- округление результата до 2 знаков после запятой
- отбрасывание незначащих конечных нулей
<br/><br/>

<a name="base-int-math"></a>
>Простая арифметика с int

Введите 2 целых числа и узнайте большее/меньшее,
сумму, разность, произведение и частное.

Особенности:
- проверка на ввод целых чисел
- округление результата до 2 знаков после запятой
- обработка исключения «деление на 0»
- преобразование int в double при делении для вывода корректного результата
<br/><br/>

<a name="sort-3-integers"></a>
>Сортировка трех целых чисел

Введите 3 целых числа и получите их вывод в порядке возрастания через запятую.

Особенности:
- проверка на ввод целых чисел
- короткий алгоритм через функцию `swap()`

Особенности:

<br/><br/>

<a name="#calc"></a>
> Калькулятор

Введите оператор и 2 операнда через пробел и узнайте результат вычисления. \
Поддерживаемые операторы: `+`, `-`, `*`, `/`, `plus`, `minus`, `mul`, `div`.

Особенности:
- проверка оператора
- проверка на ввод целых чисел и чисел с плавающей точкой
- обработка исключения «деление на 0»
<br/><br/>

<a name="financial-record"></a>
> Учет финансов

Введите количество 1-, 5-, 10-, 25-, 50-центовых монет \
и получите расчет общей суммы финансов в долларах и центах.

Особенности:
- проверка на ввод положительных целых чисел
- вывод ответа в грамматически правильной форме
- исполльзование функций
